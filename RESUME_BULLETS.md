## üé® Formatting Best Practices

### Structure

```
[Action Verb] + [What You Did] + [Tools/Technologies] + [Quantifiable Impact/Result]
```

### Example Breakdown

**Good:**
> Developed automated sales dashboard using Python, SQL, and Tableau analyzing 50,000+ transactions, reducing reporting time by 60% and enabling $2M in revenue optimizations

**Components:**
- **Action:** Developed
- **What:** automated sales dashboard
- **How:** Python, SQL, Tableau
- **Scope:** 50,000+ transactions
- **Impact:** 60% time reduction, $2M revenue opportunities

**Avoid:**
- ‚ùå "Worked on sales data"
- ‚ùå "Used SQL and Python"
- ‚ùå "Created some dashboards"

### Length Guidelines

- **Ideal:** 1-2 lines (140-180 characters)
- **Maximum:** 3 lines
- **Key rule:** Every word must add value
